# Crisisnet
CrisisNet is a multi-agent AI-driven disaster management system that coordinates crisis detection,
trust verification, resource allocation, and real-time communication to enable faster, community-led emergency response.

## Domain
OPEN INNOVATION
 - Civic Tech
 - Disaster Management 
 - AI Agents

## Problem Statement
During emergencies:
- Information is fragmented
- Response is delayed
- Volunteers are uncoordinated
- Authorities lack real-time visibility
- No single system connects citizens, volunteers, NGOs, and authorities in real time

## Solution
CrisisNet provides:
- Location-aware crisis detection
- Trust-based verification
- Dynamic task assignment
- Multi-role dashboards
- Telegram-based real-time alerts
- Agent-based orchestration between detection, trust, resource, and communication layers

## Quick Start (Run the Project)

## Prerequisites
- Node.js (v18+)
- Python (v3.9+)
- Git

## Backend
cd backend
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py

## Frontend
cd frontend
npm install
npm run dev

## Access
Frontend: http://localhost:5173
Backend API: http://localhost:8000
